Insertion Sort

Description

The insertion sort algorithm works the way how people sort a hand of playing cards. We pick an element and place it to the correct position by comparing with all the already sorted elements before this element.

  • Insertion sort wiki

Pseudocode

We use A to denote our array of elements. The pseudocode illustrates the process how we insert each element into the correct position. We assume we are going to sort the array of elements in an ascending order. 


/* Pseudocode */

Insert-Sort(A)
 n = size of A 
 for i = 2 to n
  key = A[i]
  for j = i downto 1
    if A[j - 1] > key
      A[j] = A[j-1]
  A[j] = key
