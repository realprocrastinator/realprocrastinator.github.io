Bubble Sort

Description

The bubble sort algorithm picks an element and repeatedly swaps it with the adjacent elements if they are in the wrong order, then place it to the current position.

  • Bubble sort wiki

Pseudocode

We use A to denote our array of elements, and assume we already implemented a swap function to swap two elements. The following pseudocode will sort an array of elements in ascending order. We optimize the code by using a flag to indicate whether a swapping has happened, if no swapping happended in the inner loop, then it means the array is already in an sorted order so the algorithm can terminate.


/* Pseudocode */

Bubble-Sort(A)
 n = size of A
 swapped = false 
 for i = 1 to n
  for j = i + 1 to n
    if A[j - 1] > A[j]
      swap(A[j - 1], A[j])
      swapped = true
  if swapped is false
    break
